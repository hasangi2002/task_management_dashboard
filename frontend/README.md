# Film Campaign Task Management Dashboard

## Project Overview

A modern Task Management Dashboard designed for managing Film Social Media Campaigns.

The application helps teams plan, assign, track, and monitor social media campaign tasks across multiple platforms.

## Technology Stack

### Frontend

* React.js
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js

### Database

* MySQL

### Version Control

* Git
* GitHub

---

## Features

### Dashboard

* Total Tasks Overview
* Completed Tasks
* In Progress Tasks
* Overdue Tasks
* Campaign Statistics

### Task Management

* Create Task
* Edit Task
* Delete Task
* Assign Team Members
* Due Date Tracking

### Task Attributes

* Priority (High, Medium, Low)
* Status (To Do, In Progress, Review, Completed)
* Platform (Facebook, Instagram, TikTok, YouTube)

### Analytics

* Progress Charts
* Task Completion Rate
* Platform-wise Task Distribution
* Overdue Alerts

### Additional Features

* Search Tasks
* Filter Tasks
* Responsive Design

## Installation

### Clone Repository

git clone https://github.com/hasangi2002/task_management_dashboard.git

### Frontend Setup

cd frontend

npm install

npm run dev

### Backend Setup

cd backend

npm install

npm run dev
